\# Library SOAP API



SOAP API for the library system, built with Java, Quarkus and Apache CXF.



\## Requirements



\- Java 21

\- Maven



\## Getting started



```bash

./mvnw quarkus:dev

```



The WSDL is available at:

```

http://localhost:8080/soap/LibraryService?wsdl

```



\## Testing with Postman



1\. Open Postman

2\. Click \*\*Import\*\*

3\. Select `Library\_SOAP.postman\_collection.json`

4\. All requests are ready to use — no environment setup needed
